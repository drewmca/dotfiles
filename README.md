# dotfiles

Personal configuration files for Zed editor.

## Setup

```sh
git clone git@github.com:drewmca/dotfiles.git ~/dotfiles
cd ~/dotfiles
./setup.sh
```

The setup script symlinks config files into place, backing up any existing files first.
